# weaky

A basic CouchDB/Sinatra wiki.

## Installation

1. Install CouchDB
2. Install sinatra, CouchRest, and maruku
3. Run 'ruby weaky.rb localhost:5984/weaky'

## Features

- double-bracket wiki links (\[\[link\]\])
- multiple pages with the same name allowed
